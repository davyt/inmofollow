Configuración y gestión

Todo lo que el administrador necesita saber para poner en marcha y mantener InmoFollow.

Checklist de puesta en marcha

Seguí este orden para dejar el sistema listo para los agentes:

  • Crear la empresa en Mi empresa con nombre y datos básicos
  • Completar las credenciales de WhatsApp Business (Phone Number ID y Access Token)
  • Agregar el WhatsApp Business Account ID (para ver plantillas aprobadas)
  • Crear los estados de lead que usa tu inmobiliaria (Ej: Nuevo, En seguimiento, Calificado, Cerrado)
  • Crear las plantillas de mensajes globales (primer contacto, seguimiento, etc.)
  • Crear las secuencias de seguimiento y asociarlas a estados
  • Crear los usuarios (agentes)
  • Importar los leads iniciales o cargarlos manualmente
  • Asignar los leads a los agentes
  • Configurar el cron job en el servidor (envío automático)

Configurar WhatsApp Business

Para poder enviar mensajes desde InmoFollow, necesitás tener acceso a la API de WhatsApp Cloud de Meta. Este proceso se hace una sola vez.

⚠️

Este proceso requiere una cuenta de Meta Business Manager verificada y un número de teléfono que no esté siendo usado en la app de WhatsApp. Si el número ya tiene WhatsApp instalado, hay que desvincularlo primero.

Paso a paso

1

Crear o acceder a Meta Business Manager

Entrá a Meta Business Manager con tu cuenta de Facebook. Si no tenés una cuenta de empresa, creá una.

↗ business.facebook.com
2

Crear una aplicación en Meta Developers

Entrá a Meta Developers → Mis Apps → Crear app. Seleccioná el tipo "Empresa" y completá el nombre de la app.

↗ developers.facebook.com/apps
3

Agregar el producto WhatsApp a la app

Dentro de la app recién creada, buscá el producto WhatsApp en el panel de productos y hacé clic en Configurar.

4

Agregar y verificar el número de teléfono

En WhatsApp → Configuración → Números de teléfono, agregá el número que vas a usar. Meta te enviará un código de verificación por SMS o llamada.

5

Obtener el Phone Number ID

Una vez verificado el número, en el mismo panel verás el Phone Number ID. Copialo — lo necesitás para InmoFollow.

6

Obtener el Access Token (recomendado: System User Token)

El token temporal que muestra Meta Developers vence pronto. Para producción, creá un System User en Meta Business Manager con permisos de WhatsApp y generá un token permanente.

Ruta: Meta Business Manager → Configuración del negocio → Usuarios del sistema → Agregar → Asignale el activo de WhatsApp con rol de administrador → Generar token.

↗ Usuarios del sistema
7

Obtener el WhatsApp Business Account ID

En Meta Business Manager → Configuración del negocio → Cuentas de WhatsApp, verás el ID de tu cuenta (WABA ID). Copialo también.

8

Configurar el Webhook

Para que el sistema detecte cuando un lead te responde, tenés que configurar el webhook en Meta Developers → WhatsApp → Configuración → Webhooks.

URL del webhook: https://tudominio.com/webhook/whatsapp
Token de verificación: el valor de WHATSAPP_VERIFY_TOKEN en el archivo .env del servidor.

Suscribite al evento messages para recibir los mensajes entrantes.

9

Cargar las credenciales en InmoFollow

Ingresá el Phone Number ID, el Access Token y el WABA ID en Mi empresa y guardá. Luego usá el botón "Probar conexión WhatsApp" para confirmar que todo funciona.

Plantillas aprobadas por Meta

Para poder contactar leads que nunca te escribieron (o que no respondieron hace más de 24hs), WhatsApp exige usar una plantilla aprobada previamente por Meta. Este es el requisito central para hacer outreach frío.

¿Por qué es necesario?

WhatsApp prohíbe el envío de mensajes comerciales a personas que no iniciaron la conversación, salvo que uses una plantilla aprobada. Es una medida anti-spam. Las plantillas pasan por una revisión de Meta antes de poder usarse.

Crear una plantilla en Meta Business Manager

1

Ir a Plantillas de mensajes

En Meta Business Manager → Herramientas para empresas → Gestor de WhatsApp → Plantillas de mensajes.

↗ Plantillas de mensajes
2

Crear nueva plantilla

Elegí categoría Marketing, dá un nombre en minúsculas y sin espacios (ejemplo: primer_contacto_piso) y seleccioná el idioma.

3

Escribir el mensaje con variables

Escribí el texto del mensaje. Usá {{1}}, {{2}}, etc. donde van datos variables (nombre del lead, zona, etc.).

Ejemplo: "Hola {{1}}, te contactamos desde [Tu Inmobiliaria]. Vimos que consultaste por propiedades en {{2}}. ¿Querés que te contemos las opciones disponibles?"

4

Enviar y esperar aprobación

Una vez enviada, Meta la revisa. El proceso demora entre 24 y 48 horas. Te notifican por email. El estado cambia a Aprobada.

5

Conectarla a una plantilla en InmoFollow

Una vez aprobada, andá a Plantillas de mensajes en InmoFollow, editá la plantilla correspondiente y completá la sección "Primer contacto por WhatsApp":

  • Nombre de la plantilla en Meta: exactamente igual al nombre que usaste en Meta (ej: primer_contacto_piso)
  • Idioma: el mismo que seleccionaste en Meta
  • Variables en orden: agregá las variables de InmoFollow en el mismo orden que los {{1}}, {{2}} en Meta. Si {{1}} es el nombre y {{2}} es la zona, ponés: nombre, zona
✅

Desde Mi empresa → "Ver plantillas aprobadas" podés ver en tiempo real qué plantillas están aprobadas, pendientes o rechazadas en tu cuenta de Meta.

🚫

Meta puede rechazar una plantilla si considera que el texto es demasiado promocional o genérico. Si eso pasa, hay que ajustar el texto y volver a enviarla para aprobación.

Gestión de usuarios

Accedé desde Configuración → Usuarios. Podés crear, editar y desactivar usuarios.

Crear un agente

  1. Clic en "Nuevo usuario"
  2. Completá nombre, email y contraseña inicial
  3. Seleccioná rol: Agente
  4. Guardá
  5. Compartí el email y contraseña con el agente

Cambiar contraseña de un agente

Si un agente olvidó su contraseña, desde la tabla de usuarios hacé clic en "Nueva contraseña" junto a su nombre. Ingresá la nueva contraseña y el agente puede usarla de inmediato.

Plantillas de mensajes

Las plantillas globales están disponibles para todos los agentes. Las personales solo las ve el agente que las creó.

Como administrador, podés crear todas las plantillas globales que el equipo va a necesitar: primer contacto, seguimiento a los 3 días, recordatorio de visita, etc.

Variables disponibles en el texto

VariableSe reemplaza por
{{nombre}}Nombre del lead
{{zona}}Zona de interés del lead
{{tipo_propiedad}}Tipo de propiedad que busca
{{agente}}Nombre del agente que envía

Secuencias de seguimiento

Una secuencia es una cadena de mensajes que se envían automáticamente con intervalos de tiempo definidos. Se asocian a un estado de lead (ej: la secuencia "Primer contacto" se activa cuando el lead está en estado "Nuevo").

Crear una secuencia

  1. Ir a Secuencias → Nueva secuencia
  2. Dar un nombre y asociarla a un estado de lead (opcional)
  3. Agregar pasos: cada paso tiene una plantilla de mensaje y un número de días de retraso desde el paso anterior
  4. Guardar y activar
💡

El primer mensaje de una secuencia para leads nuevos debería usar una plantilla aprobada por Meta, ya que el lead aún no habrá iniciado conversación.

Importar leads

Desde Leads → Importar podés subir un archivo CSV con leads masivos.

Requisitos del archivo CSV

  • Formato UTF-8 (para que los acentos se vean bien)
  • Primera fila con los nombres de las columnas
  • El campo nombre o name es el único requerido
  • El teléfono puede estar en formato local (09XXXXXXXX) o internacional (598XXXXXXXX)
⚠️

Antes de importar, asegurate de que el archivo tiene la columna de consentimiento de WhatsApp si los leads ya dieron su ok. Sin consentimiento, el sistema no les envía mensajes.

Asignar leads a agentes

Tenés dos opciones:

  • Individual: editá el lead y seleccioná el agente en el campo "Agente".
  • Masivo: en la tabla de leads, seleccioná varios con el checkbox, luego clic en Acciones → Asignar agente, elegí el agente y confirmá.

Configurar la inteligencia artificial

InmoFollow usa la IA de Anthropic para generar textos de plantillas. Para activarla:

  1. Creá una cuenta en console.anthropic.com
  2. Generá una API Key desde Settings → API Keys
  3. En InmoFollow, andá a Mi empresa y pegá la clave en el campo "Clave de API de IA"
  4. Guardá

A partir de ese momento, el botón "Generar con IA" estará disponible al crear o editar plantillas de mensajes.

💰

El costo de la IA se descuenta de la cuenta de Anthropic que configuraste. El uso típico para generar textos de mensajes es muy bajo. Podés ver el consumo en el dashboard de Anthropic.
